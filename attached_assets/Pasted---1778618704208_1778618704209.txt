- 파트너가 견적 제출 시 항목별 금액 + 최종금액 고정 구조

- 소장이 보는 견적 Sheet도 동일 레이아웃
- object-storage-web lib 이미 있음

### 작업 A: DB 스키마 — quotes 테이블 attachmentUrls 컬럼 추가
파일: lib/db/src/schema/quotes.ts

quotes 테이블에 컬럼 추가:
attachmentUrls: text("attachment_urls") — JSON 배열 문자열로 저장 (string[])
nullable, default null

마이그레이션 파일 생성:
lib/db/migrations/ 에 타임스탬프_add_attachment_urls_to_quotes.sql
ALTER TABLE quotes ADD COLUMN IF NOT EXISTS attachment_urls text;
