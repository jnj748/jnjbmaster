## 작업: 토스 결제창 위젯 → requestPayment 방식으로 임시 교체

### 배경
토스 실계약 심사 중으로 위젯(결제위젯) 방식이 테스트 상점에서 불가.
실계약 완료 전까지 requestPayment(일반 결제창) 방식으로 임시 교체.
실계약 완료 후 위젯 방식으로 복원 예정이므로 기존 위젯 코드는 주석으로 보존.

### 파일
artifacts/manager-app/src/pages/partner-credits.tsx

3. handleRequestTopup 함수를 requestPayment 방식으로 교체:

기존 위젯 방식 주석 처리 후 아래 로직으로 교체:

```typescript
async function handleRequestTopup() {
  if (!selectedPkgId || paying || !vendorId) return;
  const pkg = (pkgResp?.packages ?? []).find((p) => p.id === selectedPkgId);
  if (!pkg) return;
  setPaying(true);
  try {
    const clientKey = pkgResp?.tossClientKey ?? "";
    // 주문 생성
    const orderResp = await createCreditTopupOrder({ packageId: pkg.id });
    const order = orderResp.order;
    const tossClientKey = orderResp.tossClientKey ?? clientKey;

    const tossPayments = await loadTossPayments(tossClientKey);
    await tossPayments.requestPayment({
      method: "CARD",
      amount: {
        currency: "KRW",
        value: pkg.priceKrw,
      },
      orderId: order.tossOrderId,
      orderName: `관리의달인 크레딧 ${pkg.credits}C`,
      successUrl: `${window.location.origin}/partner/credits/topup/success?orderId=${order.id}`,
      failUrl: `${window.location.origin}/partner/credits/topup/fail?orderId=${order.id}`,
      customerEmail: user?.email ?? undefined,
      customerName: user?.name ?? undefined,
    });
  } catch (e: unknown) {
    setPaying(false);
    const msg = e instanceof Error ? e.message : String(e);
    if (msg.includes("PAY_PROCESS_CANCELED")) return;
    toast({ title: "결제 오류", description: msg, variant: "destructive" });
  }
}
```
