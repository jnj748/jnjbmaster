관리자 파트너 분야 관리(vendor_categories)를 SoT로 삼아 정합성 맞춤.

{ code: "building_maintenance", label: "건물관리", sortOrder: 95, active: true }

facility_maintenance 레이블을 "시설 및 영선" → "영선/수선유지" 로 변경

### 작업 B: DB 마이그레이션
lib/db/drizzle/ 에 0075_vendor_categories_align.sql 추가:

INSERT INTO vendor_categories (code, label, parent_code, sort_order, is_active)
VALUES
  ('waterproofing', '방수', NULL, 93, true),
  ('defect_diagnosis', '하자진단', NULL, 94, true),
  ('building_maintenance', '건물관리', NULL, 95, true)
ON CONFLICT (code) DO UPDATE SET
  label = EXCLUDED.label,
  is_active = EXCLUDED.is_active;

UPDATE vendor_categories
SET label = '영선/수선유지'
WHERE code = 'facility_maintenance';
